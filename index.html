Configure Integrations

Remote services

  • Hosts: SendGrid (Mail), Digital Ocean
  • Feeds: Stripe, Instagram

  1. Configure Stripe

     Set up
    • known: STRIPE_API_ENDPOINT=https://api.stripe.com/v1
    After the set up you will have:
    • STRIPE_PUBLISHABLE_KEY
    • STRIPE_SECRET_KEY
    • STRIPE_WEBHOOK_SECRET
    • Products feed (...via endpoint w/keys)

  2. Configure Instagram

     Set up
    After the set up you will have:
    • access_token - the Instagram Long lived Access (User) Token
    • client_id - the specific, shared, Instagram Application Id

  3. Configure Mailer
    Require:
    • MAIL_FROM - the address that the server itself uses. Current placeholder is: noreply@lifestylesja.com
